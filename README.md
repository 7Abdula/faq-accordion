# faq-accordion

## Getting started
### Installation

Use a package manager like [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) to install the dependencies of the project.

#### Steps using npm:

Install all dependencies:
```
npm install
```
Build tailwind.css file:
```
npm run build-css
```
Run local server:
```
npm run start
```

#### Steps using yarn:

Install all dependencies:
```
yarn install
```
Build tailwind.css file:
```
yarn run build-css
```
Run local server:
```
yarn run start
```
